## Installation

### From Source (Developer Mode)

1. Clone this repository
2. Open Chrome and go to `chrome://extensions/`
3. Enable **Developer mode** (top right)
4. Click **Load unpacked**
5. Select the `pip-plus-public` folder

## Usage

1. Open any video website (YouTube, Netflix, Bilibili, etc.)
2. Play a video
3. Press **Alt+P** (Windows/Linux) or **Ctrl+Shift+P** (Mac)
4. Or click the extension icon
